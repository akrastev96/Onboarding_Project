A Gutenberg block to display students with filtering options.

== Description ==

Students Block allows you to display students on your pages and posts with flexible filtering options:

* Set how many students to show
* Filter by status (Active/Inactive/All)
* Option to show a single student by selecting from a dropdown

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/students-block` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Run `npm install` in the plugin directory
4. Run `npm run build` to build the block assets
5. Add the "Students Block" to your page/post

== Frequently Asked Questions ==

= Does this require the Students CPT plugin? =

Yes, this block requires the Students custom post type to be registered.
